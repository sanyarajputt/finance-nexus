# SBI Nexus

An Agentic AI banking assistant built using LangGraph and Gemini.

## Features

- Multi-agent workflow using LangGraph
- Fraud Detection Agent
- Relationship Manager Agent
- Gemini-powered responses
- Transaction analysis using CSV data
- Extensible architecture for additional banking agents

## Tech Stack

- Python
- LangGraph
- LangChain
- Gemini
- Streamlit
- SQLite (Upcoming)

## Architecture

Customer → RM Agent → Fraud Agent → RM Agent → Response
